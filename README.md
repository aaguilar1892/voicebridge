## How to Run the Web App

### 1. Backend (Flask)

**Install dependencies:**

```bash
pip install -r requirements.txt
```

**Run the Flask server:**

```bash
python app.py # Windows
python3 app.py # MacOS/Linux
```
**The Flask server will run on**

```arduino
http://localhost:5001
```
